\# Спецификация мобильного приложения

\## Проект: HotelApp (Android)



\## 1. Общая информация



Мобильное приложение предназначено для поиска, просмотра и бронирования отелей.  

Приложение реализуется на платформе Android с использованием:



\- Язык: Kotlin

\- UI: Jetpack Compose

\- Архитектура: MVVM

\- Навигация: Navigation Compose

\- Работа с данными: Repository pattern (с заглушками вместо сервера)



На текущем этапе серверная часть отсутствует, данные используются локально (mock / fake data).
